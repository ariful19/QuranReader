# Workspace Instructions

- Flutter tooling root: `D:\repos\flutter\tooling`
- Flutter SDK: `D:\repos\flutter\tooling\flutter`
- Android SDK: `D:\repos\flutter\tooling\android-sdk`

## Flutter Commands

- If `flutter` is not on `PATH`, use:

- This workspace's `android/local.properties` already points to the same Flutter and Android SDK locations.

## Notes
